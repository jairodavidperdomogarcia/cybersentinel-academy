# Fundamentos Tecnológicos (Nivelación)

Antes de hackear, debes entender cómo funcionan las cosas.

## 1. Redes (TCP/IP)
*   **IP Address:** Tu DNI en la red.
*   **Puertos:** Las puertas de tu casa (80=Web, 22=SSH).
*   **Modelo OSI:**
    *   Capa 7 (Aplicación): HTTP, FTP.
    *   Capa 4 (Transporte): TCP (fiable), UDP (rápido).
    *   Capa 3 (Red): IP, Routing.

## 2. Sistemas Operativos
*   **Linux CLI:** `ls`, `cd`, `grep`, `chmod`, `chown`.
*   **Windows:** PowerShell, Event Viewer, Registro.

## 3. Cloud Basics
*   **IaaS vs PaaS vs SaaS.**
*   **Responsabilidad Compartida:** Tú aseguras tus datos, AWS asegura el datacenter.
