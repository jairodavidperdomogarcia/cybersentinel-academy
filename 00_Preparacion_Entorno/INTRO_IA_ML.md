# Introducción a IA y Machine Learning para Ciberseguridad

Conceptos clave para entender los módulos avanzados.

## ¿Qué es un Modelo?
Es un archivo matemático que ha aprendido patrones a partir de datos.
*   **Entrenamiento:** Enseñar al modelo (tarda horas/días).
*   **Inferencia:** Usar el modelo para predecir (milisegundos).

## Tipos de IA
1.  **Machine Learning (ML):** Algoritmos estadísticos (Random Forest, SVM). Bueno para datos tabulares.
2.  **Deep Learning (DL):** Redes neuronales. Bueno para imágenes y voz.
3.  **LLMs (Large Language Models):** GPT, Llama. Entienden texto y código.

## Aplicación en Seguridad
*   **Detección de Anomalías:** "Este usuario nunca se loguea a las 3 AM".
*   **Clasificación de Malware:** "Este binario se parece al ransomware WannaCry".
