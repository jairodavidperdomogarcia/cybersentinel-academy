# Proyecto Capstone: Simulación SOC Integral

**Escenario:** Global Tech Corp está bajo ataque masivo.
**Tu Misión:** Actuar como el equipo de defensa completo (Blue Team) para detener la intrusión y recuperar las operaciones.

## Fases del Proyecto
1.  **Detección:** Analizar logs de firewall y alertas de IDS.
2.  **Contención:** Identificar máquinas comprometidas y aislarlas.
3.  **Análisis Forense:** Determinar el vector de entrada (¿Phishing? ¿Vulnerabilidad Web?).
4.  **Reporte:** Escribir un informe post-incidente con recomendaciones.

Este proyecto integra todos los conocimientos del curso en un solo ejercicio práctico.
