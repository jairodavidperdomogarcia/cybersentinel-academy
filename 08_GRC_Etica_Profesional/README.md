# Gobernanza, Riesgo y Cumplimiento (GRC) y Ética

El marco legal y ético de la profesión.

## Normativas Clave
*   **ISO 27001:** Estándar internacional de gestión de seguridad.
*   **NIST CSF:** Framework de ciberseguridad (Identify, Protect, Detect, Respond, Recover).
*   **GDPR / LOPD:** Protección de datos personales.

## Ética Profesional
*   **White Hat vs Black Hat.**
*   **Reglas de Compromiso (RoE):** Nunca atacar sin permiso escrito explícito.
*   **Responsabilidad:** Reportar vulnerabilidades de forma responsable (Responsible Disclosure).
